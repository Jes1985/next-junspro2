# Miniature vidéo par défaut

Ce dossier contient les miniatures vidéo des freelances.

## Image par défaut

Pour utiliser l'image fournie comme miniature par défaut (au lieu du bloc violet), placez l'image dans ce dossier avec le nom :

`default-video-thumbnail.png`

L'image doit être au format 16:9 (ex: 1280×720px) et aux formats JPG, PNG ou WEBP.
