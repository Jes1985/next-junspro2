# Image Hero Parrainage

## Fichier requis
- **Nom**: `parrainage-hero.jpg`
- **Emplacement**: `/public/assets/img/parrainage-hero.jpg`
- **Dimensions recommandées**: 2000x1120px minimum (ratio 16:9)
- **Format**: JPG haute qualité

## Spécifications
- **Sujet**: Collaboration / Coaching / Projet / Duo qui sourit / Ambiance humaine
- **Style**: Premium, professionnel mais chaleureux
- **Contraste**: Assez lumineux pour que le texte blanc soit lisible si overlay nécessaire
- **Poids**: < 500KB (optimisé pour le web)

## Sources suggérées
- Unsplash (mots-clés: collaboration, teamwork, coaching, business meeting)
- Pexels (mots-clés: professional, team, workspace)
- Images libres de droits premium

## Note
En attendant l'image finale, un placeholder SVG est utilisé automatiquement.
