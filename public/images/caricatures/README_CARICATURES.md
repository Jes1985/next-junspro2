# Caricatures Premium pour Univers Services

## Emplacement des caricatures

Pour ajouter des caricatures haut de gamme pour chaque univers, placez vos images dans :

### Option 1 (recommandé)
```
public/images/caricatures/projects-caricature.svg
public/images/caricatures/lessons-caricature.svg
public/images/caricatures/at-home-caricature.svg
public/images/caricatures/wellnesslive-caricature.svg
public/images/caricatures/corporate-caricature.svg
public/images/caricatures/homeswap-caricature.svg
```

### Option 2
```
public/assets/img/caricatures/projects-caricature.svg
public/assets/img/caricatures/lessons-caricature.svg
public/assets/img/caricatures/at-home-caricature.svg
public/assets/img/caricatures/wellnesslive-caricature.svg
public/assets/img/caricatures/corporate-caricature.svg
public/assets/img/caricatures/homeswap-caricature.svg
```

## Formats acceptés

- **SVG** (recommandé pour qualité et légèreté)
- **PNG** (avec transparence)
- **WebP** (optimisé)

## Spécifications recommandées

- **Taille** : 120x120px minimum (sera redimensionné automatiquement)
- **Style** : Caricature haut de gamme, moderne, élégante
- **Couleurs** : Harmonisées avec la palette violet clair (#8B5CF6 → #C4B5FD)
- **Fond** : Transparent de préférence

## Thèmes suggérés pour chaque univers

1. **projects-caricature** : Personnage créatif avec outils/bureau moderne
2. **lessons-caricature** : Personnage enseignant bienveillant avec livre/tableau
3. **at-home-caricature** : Personnage relaxé dans un environnement domestique
4. **wellnesslive-caricature** : Personnage énergique en mouvement/bien-être
5. **corporate-caricature** : Personnage professionnel dans un environnement entreprise
6. **homeswap-caricature** : Personnage voyageur avec valise/maison

## Note

Si aucune caricature n'est trouvée, des illustrations SVG premium seront affichées automatiquement comme fallback.
