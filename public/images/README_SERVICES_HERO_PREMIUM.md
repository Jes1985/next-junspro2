# Photo Premium pour Services Hub

## Emplacement de la photo

Pour ajouter une photo haut de gamme sur la page `/services`, placez votre image dans l'un de ces emplacements :

### Option 1 (recommandé)
```
public/images/services-hub-hero.jpg
public/images/services-hub-hero.png
public/images/services-hub-hero.webp
```

### Option 2
```
public/assets/img/services-hub-hero.jpg
public/assets/img/services-hub-hero.png
public/assets/img/services-hub-hero.webp
```

## Spécifications recommandées

- **Format** : JPG, PNG ou WebP
- **Résolution** : Minimum 1920x1080px (Full HD)
- **Ratio** : 16:9 ou 21:9 (format paysage)
- **Taille fichier** : Optimisé (< 500KB recommandé)
- **Style** : Photo haut de gamme, professionnelle, moderne

## Suggestions de contenu

La photo devrait illustrer :
- Un environnement professionnel moderne
- Des personnes collaborant ou travaillant
- Un espace lumineux et épuré
- Des éléments visuels qui évoquent la diversité des services (projets, cours, bien-être, etc.)

## Exemples de thèmes

- Bureau moderne avec vue panoramique
- Espace de coworking élégant
- Personnes en réunion ou en formation
- Environnement de bien-être professionnel
- Architecture moderne et minimaliste

## Note

Si aucune photo n'est trouvée, un dégradé premium raffiné sera affiché automatiquement comme fallback.
