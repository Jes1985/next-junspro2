# 💾 Sauvegarde du Code Source avec Git

## 📋 Étapes pour Sauvegarder le Code Source

### 1. Ajouter tous les fichiers modifiés et nouveaux
```powershell
git add .
```

### 2. Créer un commit avec un message descriptif
```powershell
git commit -m "Sauvegarde avant modifications - Design Junspro V2"
```

### 3. Pousser vers GitHub
```powershell
git push origin main
```

---

## ✅ Vérification

Après le push, vérifiez sur GitHub que tous les fichiers sont bien présents.

---

# Se placer dans le dossier du projet

---

## ⚠️ Note

Si vous avez des fichiers sensibles (comme `.env`), assurez-vous qu'ils sont dans `.gitignore` pour ne pas les pousser sur GitHub.
