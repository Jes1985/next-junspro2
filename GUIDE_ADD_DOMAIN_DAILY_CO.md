# 📍 Guide : Où trouver "Add Domain" dans Daily.co

## 🎯 Emplacement exact

**"Add Domain"** se trouve dans **"Paramètres"** (Settings), **PAS** dans "Développeurs" !

## 📝 Étapes détaillées

### 1. Accéder au Dashboard Daily.co

1. Allez sur [https://dashboard.daily.co](https://dashboard.daily.co)
2. Connectez-vous avec vos identifiants

### 2. Naviguer vers Paramètres

Dans le **menu de gauche**, vous verrez plusieurs options :

```
┌─────────────────────────┐
│ junspro.daily.co        │
├─────────────────────────┤
│ 📊 Aperçu               │
│ 🎥 Séances              │
│ 🏠 Chambres             │
│ 📹 Enregistrements      │
│ 💳 Facturation          │
│ 💻 Développeurs  ← Vous êtes ici actuellement
│ ⚙️ Paramètres  ← C'EST ICI qu'il faut aller !
└─────────────────────────┘
```

### 3. Cliquer sur "Paramètres"

- Cliquez sur **"Paramètres"** (ou **"Settings"** si l'interface est en anglais)
- C'est l'icône avec une roue dentée ⚙️

### 4. Trouver la section "Domaines"

Dans la page Paramètres, vous verrez plusieurs sections. Faites défiler jusqu'à trouver :

- **"Domaines"** (en français)
- **"Domains"** (en anglais)
- **"Custom Domains"** (parfois)

### 5. Cliquer sur "Add Domain"

- Vous verrez un bouton **"Ajouter un domaine"** ou **"Add Domain"**
- Généralement un bouton bleu ou vert
- Cliquez dessus

### 6. Entrer votre domaine

- Une fenêtre modale ou un formulaire s'ouvrira
- Entrez votre domaine (ex: `meet.junspro.com`)
- Cliquez sur **"Ajouter"** ou **"Add"**

## ⚠️ Erreurs courantes

### ❌ "Je ne trouve pas Add Domain dans Développeurs"
- **Raison** : "Add Domain" n'est PAS dans "Développeurs"
- **Solution** : Allez dans **"Paramètres"** à la place

### ❌ "Je ne vois pas la section Domaines"
- **Raison** : Vous devez peut-être faire défiler la page
- **Solution** : Faites défiler vers le bas dans la page Paramètres

### ❌ "Le bouton Add Domain n'apparaît pas"
- **Raison** : Votre compte peut ne pas avoir accès aux domaines personnalisés
- **Solution** : Vérifiez votre plan Daily.co (certains plans gratuits peuvent avoir des limitations)

## 🔍 Si vous ne trouvez toujours pas

1. **Vérifiez votre plan Daily.co** :
   - Certains plans peuvent nécessiter une mise à niveau pour les domaines personnalisés
   - Allez dans **Facturation** pour voir votre plan actuel

2. **Contactez le support Daily.co** :
   - [https://help.daily.co](https://help.daily.co)
   - Ou utilisez le chat dans le dashboard

3. **Utilisez le domaine par défaut** :
   - Si vous n'avez pas besoin d'un domaine personnalisé, utilisez simplement `junspro.daily.co`
   - Aucune configuration supplémentaire n'est nécessaire

## ✅ Résumé rapide

1. Dashboard Daily.co → Menu gauche

3. Faites défiler jusqu'à **"Domaines"**
4. Cliquez sur **"Add Domain"**
5. Entrez votre domaine
